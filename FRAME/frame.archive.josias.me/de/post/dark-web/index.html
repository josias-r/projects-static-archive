Dark Web

Was ist das Dark Web, wofür wird es verwendet und wie erhalte ich Zugang?

Vorab: das Betreten des Dark Webs ist nicht illegal.
Das Dark Web ist, wie der Name schon sagt, die «dunkle Seite» des Internets. Im Dark Web ist es möglich, verbotene Dinge wie Waffen, Drogen und pornografisches Material zu kaufen. Viele dieser Angebote werden aber von Betrügern angeboten. Im Dark Web bezahlt man mit Kryptowährungen, damit die Zahlungen praktisch nicht nachzuverfolgen sind.
Stolpert man über eine der Seiten, auf welchen illegale Waren angeboten werden, macht man sich dadurch nicht strafbar. Erst durch das Aktivwerden im Dark Web, zum Beispiel durch den Kauf von Drogen, macht man sich strafbar. Eine Ausnahme bildet das pornografische Material. Sollte man auf eine dieser Seiten stossen, kann es sein, dass Bilder in den Cache gespeichert werden, womit man sich strafbar machen kann.

Um ins Dark Web zu gelangen benötigt man lediglich einen Tor-Browser. Ist der Browser installiert, können die Netzwerkeinstellungen vorgenommen werden. Dabei geht es darum, wie dein Zugang verschlüsselt wird und so eine sichere Verbindung aufgebaut werden kann. Danach wird eine Verbindung aufgebaut, was einige Zeit in Anspruch nimmt. Im Grunde funktioniert der Tor-Browser gleich wie ein normaler Internetbrowser, jedoch benötigt man einen spezifischen Link um die entsprechenden Websites abrufen zu können. Dafür gibt es zum Beispiel das «Hidden Wiki» welches viele solcher Links auflistet. Obwohl der Tor-Browser die IP verschlüsselt und ein Backtracking fast unmöglich macht, sollte man trotzdem vorsichtig sein. Denn vor Viren und Trojanern schützt der Tor-Browser nicht.
