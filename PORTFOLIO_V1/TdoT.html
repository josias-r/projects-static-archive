TdoT Flyer

description:

Our first «ZLO», creating a flyer. We were given all the design instructions, i.e. which colors to use. The challenge was to create an appealing flyer from ground up using only the allowed design elements. Mine turned out with a modern simplistic look.

software used:

The main software for this «ZLO» was of course InDesign. It lives up for it's purpose of creating print products. But you can't just create a document and start adding content like in Word or similar software. One part of creating a high quality print product like this flyer, is first setting up the document it self. You have to define the page site, the layout, margins, etc.

Then I was ready to start. After creating some sketches for the Design of course, I started working on the flyer. I chose a modern, simplistic look using two of the given color that were allowed. On the front page I added keywords about the event and on the back page you can find the address on a map from Romanshorn.

Also needed for small tasks was phtoshop. As you can see in the image above, I used a photography to support the identity of SBW Neue Medien. Afer taking a suitable picture, I had to set it up in photoshop for further use. I decided to go with a B/W look.
